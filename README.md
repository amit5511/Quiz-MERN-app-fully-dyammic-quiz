## About the project
Qriosity is a web application where a user can create quiz (both single correct as well as multiple correct), add negative marking to questions, attempt quiz and analyse scores.

## Techstack used
1. NodeJS
2. ReactJS
3. MongoDB
4. ExpressJS
5. Redux
